SU2

Study of the structure of the code

SU2 is an open-source suite for multiphysics simulation created by Stanford University. This code is written in C++ and have more than 500,000 lines of code. The goal of this code is to resolve partial differential equations

The core of SU2 is SU2 CFD. It can be used for compressible or turbulent flows.
