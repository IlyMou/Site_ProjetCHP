SU2

SU2 is an open-source suite for multiphysics simulations created by Stanford University. This code is written in C++ and have more than 500,000 lines of code. Meant to resolve partial differential equations on unstructured meshes, SU2 has a wide applicability in various fields, like aeronautical, automotive, naval or renewable energy. The core of the suite is SU2 CFD, which can be used for compressible or turbulent flows.

Structure of SU2 CFD

SU2 CFD is divided into several classes. Classes in CConfig enable to read the config file and configurate the simulation. Classes in COutput create output files for the simulation in the requested format. CNumerics contains the different schemes to calculate flows, but the different types of equations are computed in CSolver. The classes in CSolver are used to construct the jacobian and the source terms in each iteration. These classes also call functions from CNumerics to use scheme defined in CNumerics. However, equations are only present in their classes in CSolver, where our modifications are mainly located in a new class.
